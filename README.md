# Real-Time Epileptic Seizure Prediction using EEG Signals
Machine Learning project for real-time epileptic seizure prediction using EEG signals from the CHB-MIT dataset.


This project aims to develop a machine learning system for **real-time seizure prediction using EEG signals**.

Dataset used:
CHB-MIT Scalp EEG Database (PhysioNet)

## Project Goals

- EEG signal preprocessing
- Feature extraction
- Machine learning models for seizure prediction
- Real-time prediction pipeline

## Dataset

The dataset can be downloaded from:

https://physionet.org/content/chbmit/1.0.0/

Place the dataset in:

data/raw/
